Rationales for design of each question:
Q1 年齡 → 不直接對應假設，但它是交叉分析的基礎。你拿 Q1 去跟 Q3、Q4 做交叉，看「65歲以上的人跟其他年齡層在信任來源上有沒有差異」。如果沒差異，你的推廣策略不需要按年齡分群；如果有差異，你可能需要針對不同年齡層設計不同的觸及方式。
Q2 LINE使用基線 → 觀察「看LINE官方帳號的內容」的勾選比例。如果極低（比如低於15%），意味著你的受眾根本不習慣跟官方帳號互動，小沙彌面臨的障礙不只是 awareness，而是整個使用習慣類別都不存在——推廣計畫裡就需要加「教信眾怎麼使用官方帳號」這一步。如果比例還行，你只需要解決 awareness。
Q3 資訊管道（假設一：管道是被動的） → 看「道場LINE群組的通知」「到道場現場才知道」「法師或師姐/師兄告知」這三個被動管道的勾選總量，vs.「佛光山FB/YouTube」「官網」這些主動管道。如果被動佔壓倒性多數，你的 V3 O2O 導流邏輯成立：信眾不會自己來找小沙彌，你必須把小沙彌推到他們面前。
Q4 信任來源（假設二：法師背書是最強驅動力） → 看「法師或師父」的勾選率是不是遠高於其他選項。如果是，你 V3 裡「由主法法師於法會結束後的開示時段口頭推薦小沙彌」這個策略就有數據支撐，你跟慧讓法師提案時可以直接指著這個數字說話。如果「家人」也很高，可以在推廣設計裡加一個「鼓勵信眾回家跟家人分享」的環節。
Q5 內容偏好（假設三：信眾想要有節奏的、可重複的內容） → 看「每日一句佛法短語」的勾選率。如果高，它間接支持抽法語的方向——信眾對定期、簡短、有法味的內容有偏好。看「佛法問答」的勾選率——如果這項反而最高，意味著現有的查詢功能可能比你想的更重要，優化順序可能需要把「改善問答品質」排在「新增抽法語」前面。
Q6 開放題 → 不做結構化分析。你讀一遍，看有沒有任何你完全沒想到的東西出現。大部分會是空白或「很好」之類的無效回答，但偶爾會有一兩條讓你重新思考的。
